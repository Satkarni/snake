# snake

Weekend fun project during Covid lockdown

## Prerequisites
- ncurses  
  Install on Linux with   
  ```sh
  $ sudo apt-get install libncurses5-dev libncursesw5-dev  
  ```
  Install on Mac with   
  ```sh
  $ brew install homebrew/dupes/ncurses  
  ```
## Build
```sh
#to build  
snake$ make

#to run  
snake$ ./s

#or  
snake$ ./s1
